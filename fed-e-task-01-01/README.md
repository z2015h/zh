4. var 声明的变量可变量提升, 可在未定义的时候先调用， 在相同的作用域可以声明相同的变量； let 声明会形成一个块级作用名，在相同作用域中不能声明同样的变量，不能在未声明前调用； const 声明一个常量，跟Let具有相同的块级作用域，必须初始化，其内存指向不能改变

6. Symbol声明一个独一不二的值，  可以避免对象冲突
7. 浅拷贝只复制对象或数组第一层，可能导致指向地址是同一个，改变其中一个，另外一个也改变；深拷贝是逐级复制，不会造成指向同一个内存地址；
8. 异步编程，就是不堵塞当前线程，将一些逻辑放到回调函数中调用，eventloop是调度和监听消息队列和执行栈
